# CEHGLS: Communication-Efficient Head Gradient Linear Search for Personalized Federated Learning

This repository is released as a method-oriented companion codebase for CEHGLS.

CEHGLS is a personalized federated learning method developed on top of the design principles of [PFLlib](https://github.com/TsingZ0/PFLlib.git) [1][2]. 

CEHGLS consists of two components:

1. `HGLS` (Head Gradient Linear Search)
2. `HGC` (Head Gradient Compression)
